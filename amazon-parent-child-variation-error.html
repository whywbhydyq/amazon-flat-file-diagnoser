Parent Child

Amazon 父子体关系错误怎么修

父子体错误通常会导致整个变体组失败。先确认 parent 行、child 行和 variation_theme，再处理标题、图片和卖点。

Updated: 2026-05-30 - Independent seller utility, not affiliated with Amazon.

父体行和子体行不能按同一规则填

字段父体 parent 行子体 child 行
sku父体 SKU，例如 PARENT-TEE真实可售 SKU，例如 TEE-BLK-M
parentageparentchild
parent_sku通常留空必须指向父体 SKU
relationship_type通常留空或按模板要求通常为 variation
variation_theme填写变体主题与父体保持一致
价格 / 库存 / 商品编码通常不作为真实售卖行填写子体填写真实价格、库存和商品编码

示例父子体结构

sku	parentage	parent_sku	relationship_type	variation_theme	color_name	size_name	price	quantity
PARENT-TEE	parent			SizeColor				
TEE-BLK-M	child	PARENT-TEE	variation	SizeColor	Black	M	19.99	25
TEE-BLK-L	child	PARENT-TEE	variation	SizeColor	Black	L	19.99	18

这是示意结构，不代表所有类目都使用同一字段名。实际字段以当前类目模板和 Seller Central 上传报告为准。

Processing Report 中常见信号

错误码可能指向怎么查
99010父子体必填字段组缺值，例如 parent_child 或 variation_theme。按字段查看 variation_theme、parent_child、relationship_type。
90244颜色、尺寸、单位等变体属性值不在允许列表。回 Valid Values 检查 attribute-name。
8036变体关系创建受限或关系不被接受。确认父子体关系、账号/类目限制和是否应拆分变体。
8008关系 feed 引用的 child SKU 尚未正确创建。先创建子 SKU，再上传关系，或同批文件确认行顺序和字段完整。

修复前后示例

Before:
TEE-BLK-M	child	PARENT-TEE	variation		Black	M

After:
TEE-BLK-M	child	PARENT-TEE	variation	SizeColor	Black	M

如果 Processing Report 指向 variation_theme 缺失，先补完整父体和全部子体的 variation_theme，再检查每个子体的属性组合是否唯一。

边界说明

不同类目允许的 variation_theme 和属性组合不同；有些 ASIN 或类目可能不允许创建或修改某些变体关系。最终以 Seller Central 当前报告和目录规则为准。
