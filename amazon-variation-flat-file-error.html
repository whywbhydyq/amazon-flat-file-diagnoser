Variation Error

Amazon 变体 Flat File 错误排查

变体上传失败通常不是单纯标题或图片问题，而是父体、子体、变体主题、属性值或上传顺序没有对齐。

Updated: 2026-05-30 - Independent seller utility, not affiliated with Amazon.

变体上传失败时先看这组字段

字段组要检查什么常见错误码
关系字段parentage、parent_sku、relationship_type8008、8036、99010
变体主题variation_theme 是否父子体一致，是否是类目允许值。99010、90244、8036
变体属性颜色、尺寸、数量、单位等属性是否完整且组合唯一。90244、90112
子体售卖信息每个 child 是否有价格、库存、商品编码和必要图片。8560、8058、13013

推荐排查顺序

  1. 先确定父体是否只是分组行。父体通常不应作为真实售卖 SKU 填价格、库存、商品编码。
  2. 确认所有子体指向同一个父体。parent_sku 必须完全匹配父体 SKU，不能多空格、大小写或符号错误。
  3. 确认 variation_theme 一致。父体和全部子体应使用同一个类目允许的 variation_theme。
  4. 确认变体属性组合唯一。同一个父体下不能有两个 child 使用完全相同的颜色/尺寸组合。
  5. 用字段分组批量修复。如果报告多次指向 color_name 或 size_name，先修 Valid Values，而不是逐个 SKU 猜。
