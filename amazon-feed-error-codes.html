先用错误码判断问题类型

错误码不是最终答案，但能决定排查顺序。更准确的做法是把 error-code、error-message、attribute-name 和 SKU 放在一起看。

错误码常见含义优先检查字段指南
8541提交属性与现有 ASIN 目录信息冲突brand、title、UPC/EAN、color、size8541 指南
8560SKU 创建失败，常见于模板、字段或商品编码问题sku、product_type、external_product_id、required fields8560 指南
99010必填字段组缺值或冲突variation_theme、parent_child、relationship_type、类目必填组99010 指南
90244属性值不是该类目允许值color_name、size_name、unit_count、variation attributes90244 指南
8036变体关系创建受限或不被接受parent_sku、parentage、relationship_type、variation_theme8036 指南

20000图片或媒体 URL 无法访问main_image_url、other_image_url、SSL、DNS、robots20000 指南

错误码分组后怎么处理

  1. 阻断类先修。8541、8560、8036、99010、90244 常会阻断 SKU 创建、ASIN 匹配或变体关系。
  2. 字段类批量修。8058、90112、90244 往往能按 attribute-name 批量定位，不要逐个 SKU 手改。
  3. 依赖顺序单独看。8008、13013 可能和 SKU 是否已创建、父子体上传顺序或目录状态有关。
  4. 资源类最后复测。20000 这类图片/媒体 URL 问题需要确认 URL 公网可访问、HTTPS 和服务器响应。

示例：同一个错误码也要看字段

original-record-number	sku	error-code	error-message	attribute-name
84	CUP-BLK-12	90244	We can't accept the Count you entered for Unit Count.	unit_count
91	TEE-RED-M	90244	We can't accept the Color you entered for Color.	color_name

两行都是 90244，但一个要回 Valid Values 检查 unit_count，另一个要修 color_name。这就是首页工具需要“按字段”分组的原因。
