Amazon Inventory File Template 填写指南

很多上传错误不是字段本身难，而是模板站点、分类、版本或 Valid Values 不匹配。填写前先确认模板来源。

Updated: 2026-05-30 - Independent seller utility, not affiliated with Amazon.

填写前先确认模板来源

Inventory File Template 是用于批量添加或编辑商品的 Excel 模板。类目、站点和模板版本不对，会导致 required fields、valid values、variation theme 或 product type 校验失败。

检查项为什么重要常见错误码
站点和 marketplaceUS、UK、DE、JP 等站点的字段和值可能不同。8058、90112、90244
分类 / product type类目不同，Required 与 Valid Values 不同。8560、8058、99010
Data Definitions定义每列字段含义、是否必填、格式限制。99010、8058、90112
Valid Values颜色、尺寸、单位、材质等枚举值必须匹配允许列表。90244、90112
Template version旧模板可能缺少新必填字段或使用过期字段。8560、99010

最小填写策略

  1. 先决定是创建还是更新。创建新 SKU 和更新已有 SKU 需要的字段不同；不要把旧库存报表当作创建模板。
  2. 先填必填组，再填营销内容。优先完成 SKU、product type、brand、product-id、variation fields、price、quantity 等阻断字段。
  3. 从 Valid Values 复制允许值。不要把前台显示词、中文翻译或其它类目的值直接填进模板。
  4. 保存前保留原列头。不要改列名大小写、删除隐藏列或把 Tab 分隔文件另存为带格式的工作簿再上传。

示例：模板字段与 Processing Report 对照

original-record-number	sku	error-code	error-message	attribute-name
57	PARENT-TEE	99010	A value is missing from one or more required columns from this group: [variation_theme = null].	variation_theme
62	TEE-BLK-M	90244	We can't accept the Color you entered for Color.	color_name
81	TEE-BLK-L	8058	The value provided for a required field is invalid or missing.	size_name

处理方式：回到模板的 Data Definitions 和 Valid Values 页，分别检查 variation_theme、color_name、size_name，再回原始行号修正。

重新上传前 checklist

  • 模板来自目标站点和目标类目。
  • 父体与子体使用同一类目模板和同一 variation_theme。
  • 所有 required / conditionally required 字段组完整。
  • 枚举字段来自 Valid Values 页。
  • 日期、价格、数量、单位格式符合模板要求。
  • 上传后下载 Processing Report，而不是只看页面状态。

来源与边界

Amazon 官方说明 Inventory File Template 是用于批量添加或编辑商品的 Excel 模板；具体字段和允许值以 Seller Central 下载的当前类目模板为准。
