Inventory Template

Amazon Inventory File Template 填写指南

很多错误不是字段本身难，而是模板站点、分类或版本不对。填写前先确认模板来源。

填写前检查

  1. 确认模板来自目标站点和目标分类。
  2. 查看 Required 字段，不要只填商品基础信息。
  3. 使用 Valid Values 页中的允许值。
  4. 保存为制表符分隔文本，避免 Excel 改坏编码和列头。

上传后检查

上传后下载 Processing Report，按错误码和原始行号回到模板修复。不要直接覆盖重传整份未知错误文件。
