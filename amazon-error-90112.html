Error 90112

Amazon 90112 错误：字段值不是允许值

90112 常见于分类模板的 Valid Values。你填写的值看起来合理，但不在亚马逊当前模板允许列表中。

Updated: 2026-05-21 - Independent seller utility, not affiliated with Amazon.

常见字段

condition_type、fulfillment_channel、color_name、size_name、分类枚举属性。

修复步骤

  1. 在模板 Data Definitions 中查找该字段。
  2. 使用 Valid Values 页里的原始枚举值，不要自行翻译。
  3. 检查大小写、空格、连字符和复数形式。
  4. 如果字段没有允许值列表，先用一个 SKU 做最小测试。
